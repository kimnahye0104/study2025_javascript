식별자(=그릇)

        값을 저장, 이후 재활용.
        문자, 배열, 함수, 객체...
        (주의사항)
        1. 식별자는 숫자로 시작할 수 없다.
        2. 사용가능한 특수문자는 $, _만 쓸수 있다.
        3. break, const, var 등등 자바스크립트에서 자체로 쓰는 예약어는 사용불가.

자바스크립트 실행

태그에 script 작성

변수

        
        var 변수명 = 값;
        
        let 변수명 = 값;
        
        const 변수명 = 값;

변수타입

        기본타입 : 숫자, 문자, 불리언, null, undefined
        => 원시데이터, 단순화 할 수 없는 값

        참조타입 : 복합적인 값, 배열, 함수, 다양한 정규표현식 등, 객체...
