4차시 : 조건문

조건문 (if)

        if(조건){
            참일 때 할일;
        } else {
            거짓일 때 할일;
        }
        

        컴퓨터에서 0,1의 의미
        0 : 거짓 false
        1 : 참 true

비교연산자

        != 같지 않다면
        == 같다면
        === 타입까지 같다면
        !== 값은 같은데, 타입이 다르다면

중첩 조건문

        if(){
            if(){
            }
        }

축약 조건문

        1. 중괄호 삭제 가능
        2. if삭제후 ? 삽입. (참과 거짓은 세미콜론으로 구분.)

논리 연산자

        A && B 모두 참이면 참 반환
        A || B 하나라도 참이면 참 반환

switch

        조건에 매칭되는 케이스가 있다면, 해당 블록 코드 실행.

        switch(조건){
            case a:
                //조건의 결과가 a일때 할 일;
                break;
            case b:
                //조건의 결과가 b일때 할 일;
                break;
        }
